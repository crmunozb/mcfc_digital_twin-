# MCFC Digital Twin

Proyecto asociado al desarrollo de una infraestructura digital para adquisición, almacenamiento, procesamiento y visualización de datos experimentales de una celda de combustible.

## Estructura del proyecto

- `Dashboard/`: archivos relacionados con la visualización de datos.
- `Data/`: datos experimentales o archivos de entrada.
- `Database/`: scripts y configuración de base de datos.
- `simulador/`: código del simulador o modelo inicial.

## Estado del proyecto

Primer avance de memoria de título.
